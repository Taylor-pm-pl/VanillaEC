Additional enchants for PocketMine-MP
With easy-to-understand configuration.

# Features

- Additional enchants for PocketMine-MP
- Easy setup.

# How to Install

1. Download the latest version
2. Place the `VanillaEC.phar` file into the `plugins` folder.
3. Restart the server.
4. Done!
